# Book'em Web App

## Getting started

**Dev server**

- Minimum Node version: 22.12.0
- Recommended Node version: [22.18.0 - 24.5.0]
- Install all the neccessary packages: `npm i`
- Run on the development server: `npm run dev`
- Build: `npm run build`

**Production server**

Use `make`. The app is run as part of [book-em/infrastructure](https://github.com/book-em/infrastructure).
